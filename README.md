# Retail Sales Dashboard

## Project Overview
This Power BI dashboard analyzes retail sales data and provides insights into:
- Sales by Category
- Profit by Category
- Sales by Region
- Top 10 Sales by State

## Tools Used
- Power BI
- Microsoft Excel
- CSV Dataset

## Dataset
Sample Superstore Dataset

## Dashboard Features
- KPI Cards
- Bar Charts
- Sales Analysis
- Profit Analysis
- Regional Analysis
